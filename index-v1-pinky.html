🎯 One-Shot Modifiers (OSM) Support

What are OSM?

One-shot modifiers eliminate holding strain. Tap a modifier once, and it stays active for the next key only.

Keys marked with OSM support this feature.

How to Use

  • • Tap Alt → Tap Tab = Alt+Tab
  • • Tap Cmd → Tap C = Cmd+C
  • • Triple-tap to lock modifier
  • • No timing pressure, no holding!

Configuration: Enable in QMK with OSM(MOD_LALT) • Karabiner-Elements users: Use "sticky keys" feature
